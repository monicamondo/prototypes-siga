# Drawer Travel Soluction

## Objetivo
Protótipo independente do painel lateral de solução de viagem do módulo Conecta+.

## Tipo de componente
- Drawer lateral acionada a partir da página de origem duplicada
- Painel focado nos trechos programados para passageiros que perderam voos
- Sem navegação por tabs e sem componente de acomodações

## Stack
- HTML + CSS
- JavaScript simples para abertura e fechamento da drawer
- Design System SIGA preservado por tokens locais do protótipo
