# Drawer Conecta

## Objetivo
Reconstrução do painel lateral (drawer) do módulo Conecta+ para prototipação e iteração de melhorias de UI.

## Tipo de componente
- Drawer lateral com navegação de formulário interno
- Abertura a partir de conteúdo na página de origem

## Stack
- HTML + CSS + Bootstrap 5
- Padrão Angular 21 (estrutura de template)
- Design System SIGA (tokens via CDN)
